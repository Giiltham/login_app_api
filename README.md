# Login app API
Projet FastAPI fournissant une API de connexion.  

## Lancer le projet

Prérequis :
- python
- poetry

Utiliser l'environnement virtuel: ```poetry shell```

Installer les dépendances: ```poetry install```

Lancer le serveur: ```poetry run dev```
